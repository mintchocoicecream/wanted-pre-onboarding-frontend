# 원티드 프리온보딩 프론트엔드 - 선발 과제

## 사용 기술
- React
- React Router
- Axios
- FontAwesome
- Github Pages
  <br /><br /><br /><br />


## 프로젝트 실행 방법
  ### - 먼저, 프로젝트 실행을 위해 README 가장 아래에 있는 [Link](#link)를 클릭해 주세요.
  
  #### :: 1. 회원가입 
   - 회원가입 버튼을 눌러 회원가입을 진행해주세요.
   <img width="65%" src="https://user-images.githubusercontent.com/64584574/196025920-05db02c1-73bb-4653-822b-84b7a79537fd.PNG"/>
   
   - '@'를 포함한 이메일과 8자리 이상의 비밀번호를 입력해주세요.
   - 위 조건에 알맞은 정보를 입력한 후 '가입' 버튼을 누르면 회원가입이 완료됩니다.
   <img width="65%" src="https://user-images.githubusercontent.com/64584574/196025773-a2868c0a-1cac-4961-b46b-26e8eae04dd2.gif"/>
   <br /><br /><br /><br />
  
  
    
    
  #### :: 2. 로그인
  - 회원가입시 입력한 정보를 로그인 화면에 입력해 주세요.
  - 입력한 정보가 일치하지 않거나, 회원가입이 되어 있지 않은 경우 로그인이 진행되지 않습니다.
  <img width="65%" src="https://user-images.githubusercontent.com/64584574/196025778-d7f6d81f-4911-4d25-9fcf-bde792edd168.gif"/>
  
  - 로그인이 성공적으로 진행되면 '/todos' 화면으로 이동됩니다.
  <img width="65%" src="https://user-images.githubusercontent.com/64584574/196025782-9c857aa9-7deb-480c-8c3a-31ce72f645f0.gif"/>
  
  - 이전에 로그아웃을 진행하지 않은 채 '/'경로로 접속할 경우, '/todos' 화면으로 이동됩니다.
  <br /><br /><br /><br />
    
    
    
    
  #### :: 3. 투두(ToDo)
  - 입력창에 투두 항목을 입력한 후 '+'버튼을 누르면 투두 리스트에 추가됩니다.
  <img width="65%" src="https://user-images.githubusercontent.com/64584574/196025794-617c107a-a83a-42c2-afed-2d956384d9a3.gif"/>
  
  - 투두리스트 앞 부분의 체크박스를 클릭하면 투두 완료 상태가 됩니다.
  <img width="65%" src="https://user-images.githubusercontent.com/64584574/196025797-51c5c509-362f-4b05-a919-e284b5685091.gif"/>
  
  - 투두리스트의 연필 아이콘을 클릭하면 수정모드가 됩니다.
    - 수정모드의 입력창에 글을 입력한 후 체크 아이콘을 클릭하면 수정이 완료됩니다.
    - 수정모드의 'x'아이콘을 누르면 수정모드가 해제됩니다.
  <img width="65%" src="https://user-images.githubusercontent.com/64584574/196025789-378d06a7-aad5-43e8-add6-9c347be95ca7.gif"/>
  
  - 투두리스트의 휴지통 아이콘을 클릭하면 삭제를 위한 모달이 나타납니다.
    - 모달의 '확인'버튼을 누르면 해당 게시글이 삭제됩니다.
    - 모달의 '취소'버튼을 누르면 삭제 모달이 사라집니다.
  <img width="65%" src="https://user-images.githubusercontent.com/64584574/196025767-72f9828d-0443-4d60-a4be-76f492573020.gif"/>
  
  - 이전에 이미 입력한 투두 항목이 있을 경우, 정보를 불러와 화면에 나타냅니다.
  <br /><br /><br /><br />
 
 
 
 
  #### :: 4. 로그아웃
  - 로그아웃 버튼을 클릭하면 로그인 화면('/')으로 돌아갑니다.
  <img width="65%" src="https://user-images.githubusercontent.com/64584574/196025798-2bb97cdb-328c-4f55-b78b-2058ac3360ff.gif"/>
